# Job Board Client (React + Vite)

Modern React frontend for the Job Board platform.

## Setup

```bash
npm install
```

Create `.env`:

```bash
VITE_API_URL=http://localhost:8000/api
```

Run:

```bash
npm run dev
```

## Features implemented

- Modern SaaS UI with Tailwind + reusable UI components
- Job cards with badges, salary badge, company logo
- Job details page with requirements + resume upload in apply form
- Saved jobs (bookmark) + saved jobs page
- Upgraded filters (salary/work mode/experience)
- Company dashboard improvements (applicant counts)
- Admin dashboard charts (timeseries)
- Dark mode toggle
